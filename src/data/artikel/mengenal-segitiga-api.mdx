Nyala api akan terbentuk ketika tiga unsur tersebut terpenuhi dengan jumlah yang cukup. Api akan terus menyala selama jumlah atau konsentrasi ketiga elemen api tersebut mencukupi. Api akan padam ketika salah satu unsur hilang atau habis.

Sesuai dengan teori segitiga api, syarat terbentuknya api adalah harus ada tiga unsur kimia yang bereaksi. Tiga unsur api tersebut adalah oksigen, bahan bakar, serta sumber panas. Lalu, bagaimana proses terbentuknya segitiga api? Teori segitiga api menjelaskan mengenai hal ini.

Api terbentuk dari reaksi oksidasi yang melibatkan gas oksigen. Oksigen dibutuhkan pada reaksi pembakaran, sedangkan bahan bakar akan mendukung reaksi pembakaran tersebut. Sumber panas berperan untuk menaikkan bahan bakar ke suhu pengapiannya.

Reaksi oksidasi yang menghasilkan api akan terus berlangsung ketika konsentrasi bahan bakar dan oksigen masih mencukupi dan suhunya masih mendukung. Menurut teori segitiga api, api baru bisa padam jika salah satu unsur api hilang atau berkurang.

## 3 UNSUR YANG MEMBENTUK SEGITIGA API

Segitiga api adalah reaksi kimia dari tiga unsur yang menghasilkan nyala api. Segitiga api apa saja? Ada tiga unsur pada segitiga api, yaitu oksigen, bahan bakar, dan sumber panas. Bagaimana unsur segitiga api tersebut bisa membentuk api? Berikut penjelasannya.

### 1. OKSIGEN

Unsur segitiga api yang pertama adalah oksigen (O2). Udara normal atau atmoster bumi mengandung sekitar 21% oksigen. Untuk membentuk nyala api, hanya dibutuhkan sekitar 16% oksigen saja.

### 2. SUMBER PANAS

Selain oksigen, teori segitiga api juga menyatakan bahwa api terbentuk karena adanya sumber panas. Temperatur yang tinggi akan membuat risiko kebakaran semakin tinggi. Suhu panas bisa berasal dari beberapa macam sumber.

### 3. BAHAN BAKAR

Teori segitiga api menyatakan bahwa api tidak akan terbentuk tanpa adanya bahan bakar. Sesuai namanya, bahan bakar merupakan berbagai material yang bersifat mudah terbakar. Jenis bahan bakar sangat beragam, baik benda cair, padat, atau gas.
